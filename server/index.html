Travel Starter Backend API

PATCH /itin/:id

Adds a new place to an existing itinerary.

Needs the _.id passed along.

PATCH /itin/item/:id

Deletes a place on an existing itinerary.

Needs the _.id and place passed along.

PATCH /itin/publish/:id

Publishes an itinerary draft.

Needs the _.id and itinerary passed along.

GET /itin/:id

Fetches a specific itin.

Needs the _.id passed along.

DELETE /itin/:id

Deletes a specific itin.

Needs the _.id passed along.

POST /itin/create

Creates a new itin.

Needs the userId passed along.

GET /itin/all

Fetches all itins in the db.

Nothing needs to be passed through.

POST /itin/mine/done

Fetches all the finished itins of a user.

Needs the userId passed along.

POST /itin/mine/drafts

Fetches all the draft itins of a user.

Needs the userId passed along.
