POST /itin/create

Creates a new itin.

Needs the userId passed along for it to work.

PATCH /itin/:id

Adds a new place to an existing itinerary.

Data object is already formatted in card.js

GET /itin/all

Fetches all itins in the db.

Nothing needs to be passed through.

GET /itin/:id

Fetches a specidic itin.

_id needs to be passed through.

POST /itin/mine

Fetches all the itins of a user.

Needs the userId passed along for it to work.
